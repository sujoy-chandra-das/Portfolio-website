Talent Acquisition System (TAS)

The Talent Acquisition System is a comprehensive recruitment platform developed to streamline the hiring process for Southeast University.

Features

User-Friendly Interface

Designed for both job applicants and HR professionals.

Secure Login

Implementing secure user registration and login functionality.

Applicant Dashboard

Manage applications and view application statuses easily.

Job Posting

HR managers can post job openings seamlessly.

Guidance on How to Apply

Step-by-step guidance for applicants on the application process.

Technologies Used

Angular

Framework for building the front-end of the application.

Spring Boot

Framework used for building the back-end services.

Tailwind CSS

Utility-first CSS framework for styling the application.

HTML

Markup language used for structuring web content.

Java

Programming language used in back-end development.
