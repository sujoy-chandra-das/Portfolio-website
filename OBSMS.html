Online-Based Store Management System (OBSMS)

The Online-Based Store Management System (OBSMS) is a comprehensive web application designed to manage various aspects of an online store. This project was developed to enhance practical skills in web development, covering both frontend and backend functionalities.

Features

Authentication & User Management

✅ Admin Login (via login.php)
✅ Session Handling (via header.php)
✅ Admin Logout (via logout.php)

Product & Stock Management

✅ View Stock (via index.php)
✅ Update Stock (Edit product details)
✅ Delete Products

Purchase Management

✅ Purchase Form (via purchase.php)
✅ Insert Purchased Products (Stored in product and purchase tables)

Sales Management

✅ Sell Products (via sales.php)
✅ Update Stock After Sale
✅ Record Sales in sales Table

Reports & Analytics

✅ Purchase Reports (via purchaseReport.php, filtered by date)
✅ Sales Reports (via salesReport.php, filtered by date)

User Interface (UI) Features

✅ Navbar with Authentication Check
✅ Cart UI with Add/Remove Functionality
✅ Responsive Layout using Tailwind CSS

Technologies Used

Frontend

✅ HTML – Structure of the web pages
✅ Tailwind CSS – Styling and responsive design
✅ JavaScript (Vanilla/Optional) – For interactivity

Backend

✅ PHP – Server-side logic and database interactions

Tailwind CSS

✅ Utility-first CSS framework for styling the application.

HTML

✅ Markup language used for structuring web content.

Database

✅ MySQL – Storing product details, sales, purchases, and user data

Other Technologies & Features

✅ Session Management – Using PHP for authentication and user sessions
✅ CRUD Operations – Add, update, delete, and retrieve product data
✅ Reporting System – Generates purchase and sales reports
✅ Authentication – Secure login/logout system for admins
✅ Stock Management – Tracks product stock levels
