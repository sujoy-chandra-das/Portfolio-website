Exam Portal API

Project Description: A Spring Boot-based RESTful API for managing and conducting online quizzes. Enables quiz creation, question management, secure access, and result evaluation.

Features

User Authentication

JWT-based authentication system with role-based access control for Admin and Users.

Quiz Management

Admins can create, update, and delete quizzes and assign questions to them.

Question Bank

Store and organize multiple-choice questions by categories and topics.

Timed Quiz Attempts

Users can take quizzes within a set time limit. The system records and evaluates submissions.

Result Evaluation

Automatic scoring and result generation with correct/incorrect answer tracking.

Category-Based Filtering

Allows filtering and organizing quizzes based on subject or difficulty level.

Technologies Used

Spring Boot

Back-end framework used for building RESTful services quickly and securely.

Java

Core programming language used to build the application logic and services.

MySQL

Relational database used to persist quizzes, questions, users, and results.

Spring Security

Used for securing endpoints with JWT and managing user roles and permissions.

Postman

Used for API testing and validating endpoints during development.

Maven

Project management and dependency tool for building the Spring Boot application.
